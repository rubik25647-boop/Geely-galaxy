# Geely Galaxy Kazakhstan

Статический сайт официального представителя Geely Galaxy в Казахстане.

## Страницы

- `index.html` — главная (модели, новости, о компании Allur)
- `model-ex5.html` — Geely Galaxy EX5 (электро SUV)
- `model-ex5-emi.html` — Geely Galaxy EX5 EM-i (гибрид E-DHT)
- `model-cityray.html` — Geely Galaxy Cityray (хэтчбек)
- `model-preface.html` — Geely Galaxy Preface (седан)
- `dealers.html` — дилеры (14 точек по Казахстану)
- `offers.html` — спецпредложения
- `owners.html` — владельцам (гарантия, запчасти, поддержка)
- `buyers.html` — покупателям
- `test-drive.html` — заявка на тест-драйв

## Стек

Чистый HTML + CSS + JS, без сборки. Шрифт GeelyGalaxy (OTF/TTF в `fonts/`), иллюстрации в `images/`.

## Локальный запуск

Откройте `index.html` в браузере или запустите любой статический сервер:

```bash
python -m http.server 8000
```

## Деплой

Сайт статический — подойдёт GitHub Pages, Netlify, Vercel или любой хостинг.
